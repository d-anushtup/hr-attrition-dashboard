# HR Attrition Dashboard

## Overview

This project analyzes employee attrition patterns to identify key drivers of workforce turnover.

## Tools Used

* Tableau Public
* CSV dataset

## Key Questions

* Which departments have the highest attrition?
* Which age groups are most affected?
* How does job role influence attrition?

## Key Insights

* Sales roles show disproportionately high attrition
* Mid-career employees (25–34) are most likely to leave
* Attrition is concentrated in specific job roles rather than evenly distributed
